# Tab Manager (Sekme Yöneticisi)

**Tab Manager**, tarayıcınızda açık olan çok sayıda sekmeyi alan adlarına (domain) göre gruplayarak yönetmenizi sağlayan, bellek kullanımını azaltmanıza yardımcı olan bir Chrome eklentisidir.

## Özellikler

- **Otomatik Gruplandırma:** Açık olan tüm sekmeleri web sitelerine (domain) göre otomatik olarak kategorize eder.
- **İstatistikler:** Toplam açık sekme sayısını anlık olarak gösterir.
- **Toplu Kapatma:** Tek bir tıklama ile belirli bir sitedeki tüm sekmeleri kapatabilme imkanı.
- **Akıllı Favicon Desteği:** Sekme ikonlarını çeker, yüklenemeyen ikonlar için Google Favicon servisini yedek olarak kullanır.
- **Sıralı Liste:** En çok sekme barındıran siteleri en üstte göstererek önceliklendirir.

## Kurulum

1. Bu projeyi bilgisayarınıza indirin veya klonlayın.
2. Chrome tarayıcısında `chrome://extensions/` adresine gidin.
3. Sağ üst köşedeki **"Geliştirici Modu"**nu aktif hale getirin.
4. **"Paketlenmemiş öğe yükle"** butonuna tıklayın.
5. Proje klasörünü (`/tab-manager`) seçin.

## Teknolojiler

- JavaScript (ES6+)
- Chrome Tabs API
- CSS3 & Flexbox
